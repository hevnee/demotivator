# Демотивационный генератор

[English](README.md) | [Russian](RU_README.md)

## Обзор

Демотивационный генератор - это GUI приложение которое поможет создать забавное изображение с подписью под изображением в формате "постер" или "демотиватор"

## О проекте

Этот проект был создан с помощью Python используя такие зависимости как: PySide6 и Pillow.

Я сделал его для демонстрации своих знаний. Код реализован чисто (возможно не настолько чисто как вы думаете, но не грязно) и я хотел показать этим что я научился за 4 месяца

## Установка

1. Установка через релизы

    * Перейдите в раздел [Releases](https://github.com/hevnee/demotivator/releases/latest) репозитория
    * Скачайте последнюю версию демотиватора
    * Распакуйте архив и запустите приложение

2. Установка через git

    Требования: Python 3.11

    ```bash
    git clone https://github.com/hevnee/demotivator.git
    cd demotivator

    pip install -r requirements.txt

    python main.py
    ```
